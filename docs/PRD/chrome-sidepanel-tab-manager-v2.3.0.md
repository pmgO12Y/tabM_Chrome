# Chrome Sidepanel Tab Manager v2.3.0

- 版本号：`2.3.0`
- 日期：`2026-03-28`
- 修改类型：`MINOR`

## 修改内容
- 新增侧边栏向下滚动时的双吸顶能力。
- 当前窗口标题滚出顶部后，窗口头会固定在侧边栏顶部。
- 当前网页组标题滚到窗口头下沿后，组头会固定在窗口头下面。
- 新增窗口头和组头的 `push-off` 顶替逻辑，切到下一个窗口或组时，旧吸顶头会被平滑顶走。
- 吸顶头保留现有点击展开/收起行为。

## 修改原因
- 用户希望在侧边栏内容较长时，向下滚动仍能清楚知道自己当前处于哪个窗口、哪个网页组。
- 当前列表滚动到底部后，窗口和组的上下文会丢失，辨识成本较高。
- 该需求属于新的滚动交互增强，不只是样式微调。
