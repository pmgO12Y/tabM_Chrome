# Chrome 侧边栏标签管理插件 PRD v2.1.21

## Executive Summary
- 本版本在 `v2.1.20` 的基础上，进一步收小网页组标题条的圆角。
- 目标是让组标题视觉更利落，顶部圆角从较大的圆角改为 `2px`。

## Problem Statement
- 当前网页组标题条的圆角仍然偏大，不符合用户希望“调成圆角 2”的视觉要求。

## Target Users
- 希望侧边栏网页组标题风格更克制、更接近直角卡片的用户。

## Solution Overview
- 在 `styles.css` 中将网页组标题条圆角调整为：
  - 展开态：`2px 2px 0 0`
  - 折叠态：`2px`
- 不修改组内容区、颜色、动画和交互逻辑，只收小标题条圆角。

## Success Metrics
- 网页组标题条顶部圆角明显变小。
- 展开态和折叠态的圆角表现保持一致，不出现状态切换时风格不统一。

## User Stories / Acceptance Criteria
- 作为用户，我希望网页组标题条圆角更小，看起来更干净。
- 作为用户，我希望展开和折叠时圆角规则一致。

## Out of Scope
- 本版本不调整组内容区底部圆角。
- 本版本不修改窗口行、标签行样式和动画逻辑。

## Risks / Dependencies
- 本次仅改标题条圆角，不触碰其他布局结构，风险极低。

## Open Questions
- 当前无新的产品决策问题。
