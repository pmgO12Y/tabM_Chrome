# Chrome 侧边栏标签管理插件 PRD v2.0.21

## Executive Summary
- 本版本按用户指定值统一放大侧边栏文字字号。
- 目标是让网页标题、窗口标题和右侧辅助信息都更清晰易读。

## Problem Statement
- 用户明确给出了新的字号要求，需要统一调整主文字和右侧辅助信息的大小。

## Target Users
- 希望侧边栏文字更大、更清晰的用户。

## Solution Overview
- 将网页标题字号改为 `16px`。
- 将窗口标题行主文字字号改为 `16px`。
- 将右侧“当前”字号改为 `14px`。
- 将右侧数量字号改为 `14px`。

## Success Metrics
- 网页标题显示为 `16px`。
- 窗口标题行主文字显示为 `16px`。
- 右侧“当前”和数量都显示为 `14px`。

## User Stories / Acceptance Criteria
- 作为用户，我希望网页标题更大，改为 `16px`。
- 作为用户，我希望窗口标题更大，改为 `16px`。
- 作为用户，我希望右侧“当前”和数量统一改为 `14px`。

## Out of Scope
- 本版本不改布局逻辑、不改交互逻辑、不改颜色和排序。

## Risks / Dependencies
- 字号变大后，如果后续继续收紧布局，可能需要再微调行高和间距。

## Open Questions
- 当前无新的产品决策问题。
