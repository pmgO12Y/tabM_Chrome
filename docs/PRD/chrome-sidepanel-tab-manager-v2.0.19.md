# Chrome 侧边栏标签管理插件 PRD v2.0.19

## Executive Summary
- 本版本微调当前选中网页的左侧竖条高度。
- 目标是让竖条更接近整条选中底色的高度，上下视觉更齐平。

## Problem Statement
- 当前选中竖条只有 `16px` 高度，相比整条蓝色选中底明显偏短。
- 用户要求把竖条调高，与选中状态底色齐高。

## Target Users
- 对选中态视觉完整性有要求的用户。

## Solution Overview
- 将当前选中竖条从垂直居中短条，改为贴近整行高度的长条。
- 使用固定上下内缩，让竖条与选中底色上下边界基本对齐。

## Success Metrics
- 竖条高度显著增加。
- 竖条与选中底色上下更齐平，不再显得孤零零一小段。

## User Stories / Acceptance Criteria
- 作为用户，我希望当前选中竖条更高。
- 作为用户，我希望竖条和整条选中底色看起来属于同一组选中态。

## Out of Scope
- 本版本不改颜色、不改宽度、不改交互逻辑。

## Risks / Dependencies
- 当前无额外风险，本次是单点样式微调。

## Open Questions
- 当前无新的产品决策问题。
