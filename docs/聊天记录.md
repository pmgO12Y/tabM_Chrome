## 2026-04-07 IconPark 图标替换 + 悬浮提示

### 用户需求
1. 使用 https://iconpark.oceanengine.com/official 中的 icon（IconPark 官方图标库）
2. icon 应该在鼠标悬浮时显示按钮名或功能介绍

### 分析
- 项目已经在使用 `@icon-park/react`（IconPark 官方 React 组件库），图标来源正确
- 工具栏按钮（`SidepanelToolbar.tsx`）已有悬浮提示 ✅
- **问题**：搜索框中的图标（清除、模式切换、移动到新窗口）使用的是**内联 SVG**，不是 IconPark 组件，也没有统一风格的悬浮提示

### 修改内容
#### SearchBar.tsx 完全重写
- 移除所有内联 SVG，替换为 IconPark 组件：
  - 搜索图标 → `<Search theme="outline" size="14" />`
  - 清除按钮 → `<CloseSmall theme="outline" size="14" />`
  - 过滤模式 → `<Filter theme="outline" size="14" />`
  - 高亮模式 → `<Star theme="outline" size="14" />`
  - 移动到新窗口 → `<MonitorOne theme="outline" size="14" />`
- 三个按钮（清除、模式切换、移动）都添加了统一的悬浮提示：
  - 清除搜索：`"清除搜索"`
  - 模式切换：`"当前：过滤模式"` 或 `"当前：高亮模式"`
  - 移动到新窗口：`"移动到新窗口"`
- 提示文字风格与工具栏保持一致（使用 `calculateToolbarTooltipPlacement` 计算位置）

#### styles.css 更新
- 移除 `.search-bar__clear`、`.search-bar__mode-toggle`、`.search-bar__move-button` 等分立样式
- 统一为 `.search-bar__button` 样式（28x28px，圆角，hover 效果）
- 新增 `.search-bar-tooltip` 样式（深色背景、白色文字、淡入动画）
- 图标 SVG 使用 `!important` 覆盖 IconPark 的内联宽高
